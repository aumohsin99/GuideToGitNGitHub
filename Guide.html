Brief Introducation

Let's breifly explain about the terms we are going to explore.

Git

It is a version control system. You can manage different versions of your code using this tool. More importantly, you can create different branches of code and revert to previous stable version.

GitHub

It is an online tool for code management. You can use this site to save your code and projects. Later on, you can retrieve your project from here to your local desktop PC. It provides you a number of options to collaborate with your team and organization.
